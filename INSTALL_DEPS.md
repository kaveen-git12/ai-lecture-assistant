# Upgrade Package Dependencies

Run this command to install multer (required for file uploads):

```bash
npm install multer --save
```

Or install all packages fresh:

```bash
npm install
```

This will install all dependencies including:
- multer (file uploads)
- mongoose (MongoDB)
- JsonWebToken (JWT)
- OpenAI (API client)
- bcryptjs (password hashing)
- pdfkit (PDF generation)
- docx (Word documents)
- And all others listed in package.json
